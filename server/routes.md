# Node backend for MySQL

Run from the `server` folder:

- `npm install`
- `npm start`

Backend endpoints:
- `GET /api/students`
- `POST /api/students`
- `PUT /api/students/:id`
- `DELETE /api/students/:id`
- `GET /api/attendance`
- `POST /api/attendance`
- `PUT /api/attendance/:id`
- `DELETE /api/attendance/:id`
- `GET /api/marks`
- `POST /api/marks`
- `PUT /api/marks/:id`
- `DELETE /api/marks/:id`
- `GET /api/timings`
- `POST /api/timings`
- `PUT /api/timings/:id`
- `DELETE /api/timings/:id`
- `GET /api/fees`
- `POST /api/fees`
- `PUT /api/fees/:id`
- `DELETE /api/fees/:id`
- `GET /api/settings`
- `PUT /api/settings`
- `GET /api/holidays`
